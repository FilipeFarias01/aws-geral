🌐 AWS GERAL
📌 O que é a AWS

A Amazon Web Services (AWS) é uma plataforma de computação em nuvem criada pela Amazon que oferece diversos serviços como armazenamento, servidores, banco de dados, inteligência artificial, entre outros.

Ela permite que empresas e desenvolvedores utilizem infraestrutura de tecnologia sem precisar comprar ou manter servidores físicos, pagando apenas pelo uso.

🌍 Infraestrutura Global

A AWS possui uma infraestrutura global distribuída para garantir alta disponibilidade, desempenho e segurança.

🗺️ Regiões (Regions)

As Regiões são áreas geográficas espalhadas pelo mundo onde a AWS possui data centers. Cada região é independente e localizada em diferentes países ou continentes.

Exemplo:

América do Sul (São Paulo)
América do Norte
Europa
Ásia
🏢 Zonas de Disponibilidade (Availability Zones)

Dentro de cada região existem várias Zonas de Disponibilidade, que são conjuntos de data centers isolados fisicamente.

Elas funcionam assim:

Cada zona opera de forma independente
Estão conectadas por redes de alta velocidade
Garantem que, se uma falhar, as outras continuam funcionando

Isso aumenta a confiabilidade dos sistemas.

💰 Economia da Nuvem
⚙️ Modelo Pay-as-you-go

A AWS utiliza o modelo Pay-as-you-go (pague conforme o uso).

Ou seja:

Você paga apenas pelos recursos que utilizar
Não precisa investir em infraestrutura antecipadamente
Pode escalar os serviços conforme a necessidade

Exemplo:
Se você usar um servidor por 2 horas, paga só por essas 2 horas.

📈 Economia de Escala

A AWS consegue oferecer preços competitivos por causa da economia de escala.

Isso funciona assim:

A AWS atende milhões de clientes
Compra e mantém infraestrutura em grande escala
Reduz custos operacionais
Repassa parte dessa economia para os clientes

Resultado:
Serviços mais baratos e acessíveis do que manter servidores próprios.

🚀 Conclusão

A AWS facilita o acesso à tecnologia, permitindo que empresas de qualquer tamanho utilizem recursos avançados com baixo custo inicial, alta escalabilidade e grande confiabilidade.
